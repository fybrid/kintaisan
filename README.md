# 勤怠さん

## 使用技術一覧
- HTML
- CSS
- JavaScript
- Java
  JDK21以上（VScodeの拡張機能を使用するなら）
- Springboot
- PostgreSQL (DB)

## クイックスタート
1. 以下のコマンドを実行
```shell
$ cp .env.example .env
```
2. PostgresQLで新しくDBを作成
3. `.env`内の`DB_USERNAME`と`DB_PASSWORD`の値を設定
4. [メイン関数](./src/main/java/com/kyosaka/kintaisan/KintaisanApplication.java)を実行
